# Contributing to Mist API Info

1. Fork the repository and clone your fork.
2. Create a branch (`git checkout -b feature/your-feature`).
3. Install deps with `pip install -e .` and run `python -m pytest -q`.
4. Keep code under `src/mist_api_info/` and tests under `tests/`.
5. Update docs/README/CHANGELOG when behaviour changes.
6. Commit (`git commit -m "feat: concise summary"`) and push.
7. Open a PR against `main` with test results.
